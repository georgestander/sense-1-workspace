# Sense-1 Workspace v1 Roadmap

## Current baseline: v0.7.1

This baseline tracks the latest shipped desktop shell version. Each release should
keep this roadmap marker, `desktop/package.json`, and the matching changelog entry
in sync.

## v1 focus

- Keep live desktop sessions responsive during long-running agent output.
- Tighten the thread composer so the primary controls stay simple and legible.
- Ship incremental desktop reliability work before expanding the product contract.
